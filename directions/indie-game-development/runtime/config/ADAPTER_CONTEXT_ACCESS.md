# Adapter Context Access

## Adapters

- ChatGPT: planning/synthesis/work chats/verification
- Codex: bounded repository persistence only with handoff
- GitHub: exact repo/path/ref source reads
- human actions: explicit decision/confirmation
- old Direction files: request-only legacy_evidence

## Boundary

Adapters cannot create accepted state.

This clean_start package is for one Direction only. Old Direction files are legacy_evidence only and are not accepted_v3_state.

No actual ChatGPT Project UI update is performed. No Project Files/Sources refresh is performed. Project Files/Sources are cache/context only. Runtime Console is read-only.
