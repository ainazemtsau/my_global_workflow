# Project Питание Portable Setup Bundle

This folder is the portable setup bundle for ChatGPT Project `Питание`.

Use the files in this order:

1. Paste `PROJECT_INSTRUCTIONS.md` into ChatGPT Project Instructions.
2. Upload `PITANIE_PROJECT_SOURCE.md` as the required Project Source.
3. Paste `BOOTSTRAP_PROMPT.md` into the first chat.
4. Use `VALIDATION_PROMPTS.md` for dry-run checks.

The required source upload set is exactly one file: `PITANIE_PROJECT_SOURCE.md`.

Optional real nutrition files can be uploaded after setup, including prior menus, preferences, equipment notes, shopping notes, or previous carryover blocks.

`PROJECT_PITANIE_SETUP_PACKAGE.md` is only the package index/manifest. The archived pre-repair monolith is preserved at `archive/PROJECT_PITANIE_SETUP_PACKAGE_MONOLITH_2026-05-18.md`.
