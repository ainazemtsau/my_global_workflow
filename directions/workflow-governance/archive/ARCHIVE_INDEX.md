# Workflow Governance Archive Index

`archive/` is inert storage. It is not accepted proof state.

`archive/results/` is for future inert result snapshots if explicitly saved.

`archive/legacy-vnext-r/` is for legacy evidence only, not active state. Any preserved old Workflow Governance material under this path is `legacy_evidence_only`.

Legacy material may become accepted state only through Legacy Import Receipt, Verify, and Commit.

Do not restore old vNext-R files unless explicitly needed for preservation. If restored, place them only under `archive/legacy-vnext-r/` and label them `legacy_evidence_only`.
