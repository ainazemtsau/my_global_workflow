Important concept anchors:

- EXOCORTEX is a personal digital brain, not a chat wrapper.
- The base model is an upgradable cognitive engine.
- Durable value lives in persistent memory, context, tools, workspace surfaces, skills, feedback loops, and Reality Alignment.
- Context Compiler prevents dumping the whole user world into prompt context.
- Memory is layered: working, episodic, semantic, procedural, decision, environment, archive/RAG.
- The system should learn from outcomes / prediction error.
- Reality Alignment prevents blind agreement and protects against scope creep and bad decisions.
- Tools/actions are the system body.
- Workspace surfaces are extensible environments, not isolated UI features.
- Max Vision and Min Proof must stay separate.
