# Phase Working Context — Personal Workflow App Kernel Exploration

## User clarification

The project is development of the current personal Workflow. Today it operates through ChatGPT, GitHub, Codex, and related tools. The possible app/kernel is not a separate commercial roadmap by default. It is a way to explore whether the current Workflow should become a more custom, extensible personal system.

The user controls pace, scope, and decisions.

## Concept seed

The EXOCORTEX document describes a possible personal AI companion / external brain with memory, tools, initiative, learning from prediction error, Reality Alignment, working rooms, dashboards, widgets, automations, and strategic support.

For this Phase, the document is directional input only.

## Working interpretation

The strongest near-term framing is not "build an app." It is:

A persistent personal operating loop that observes user work, compiles relevant context, remembers decisions/patterns, detects mismatch, and applies Reality Alignment to improve future decisions/actions.

## P0 scope boundary

Do not turn the concept into a roadmap. Do not design architecture. Do not start implementation. Do not research broadly until G1 defines a narrow Goal and evidence gap.
