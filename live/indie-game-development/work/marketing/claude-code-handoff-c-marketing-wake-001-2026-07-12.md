## B1 research checkpoint — стратегия каналов

Frozen закон остаётся: «мы никто», capture-first, реальный gameplay clip — главный рычаг, не строить treadmill.

Исследовательский вывод, предварительно одобренный владельцем словами «Так, окей» с уточнением, что показывать пока нечего:

1. **Incident Pulse — первый тест после реального capture.** Три разных настоящих игровых инцидента за 4–6 недель; native video, один релевантный subreddit за раз, Steam UTM + дневной wishlist delta + player-intent comments. Стартовый kill: после трёх попыток суммарно `<50` trusted Steam visits, нет заметного wishlist lift и нет запросов игроков → остановить до качественно нового материала.
2. **Creator Crew — главный рычаг после стабильного co-op demo.** Pilot 40 строго совпадающих creators; `<2` ответов и `<3` key activations → переделать hook/pitch/demo. При проходе расширить до 150, максимум 300 без оплаты; после 150 и ≤1 материала без измеримого эффекта → заморозить.
3. **Shorts/devlog — только производные.** Два батча по три ролика из уже снятого материала; raw views не считать успехом. Нет Steam/creator reuse либо >30 минут человеческого труда на единицу → прекратить.
4. Steam Coming Soon + UTM = capture substrate, не отдельная стратегия. Next Fest = условный усилитель; регистрация обратима, участие только при качественном demo.
5. X/Bluesky/Instagram — passive handle only; регулярный devlog, ежедневный постинг и paid ads отсечены.

Критические обновления старого research:

- Site-wide Reddit «10% rule» больше не авторитет: действует текущая spam policy + правила конкретного community.
- Не бросать рекламу в r/Barotrauma/r/SS13: их правила требуют прямой связи с этими играми. r/SurvivalGaming явно допускает indie promo; r/SurvivalHorror требует modmail; r/IndieGames запрещает store links; r/IndieGaming требует историю аккаунта и широкое GenAI disclosure.
- YouTube Shorts descriptions/comments have non-clickable URLs; raw Shorts views changed and are vanity-prone. Use engaged views/viewed-vs-swiped only as creative diagnostics.
- Current Steam Next Fest first days are broadly randomized, then personalized by player behavior; own livestream is optional.
