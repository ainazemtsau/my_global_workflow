# Вопрос: AI-конвейер ассетов (картинки + голос) + капсула + проверка

Порождён q-foundation. Решает КАК практически генерить ассеты под стиль игры, КАК проверять (без «спросить своих») и КОГДА доплачивать человеку. Содержание форжим внутри — здесь только рамка.

## Что решить
- **Лестница генерации:** ChatGPT Pro (подробный промпт) СНАЧАЛА → локальная дообучка на своих low-poly рендерах только если промпт не попадает в стиль.
- **Style-lock** под собственный вид игры + грунтование реальными скринами + человеко-финиш (Movavi).
- **Проверка лучше «спросить своих»:** скептик-протокол (слепой A/B vs референс-капсула + скрытый зонд «первое впечатление / что оттолкнёт» со счётом спонтанных «AI/слоп» + прямой пост в нишу, который ПЕРЕВЕШИВАЕТ опрос); порог по 2-из-3 конвергенции, калибровать на доверенной капсуле; бюджет ~$60–150/цикл, кап ~3 цикла.
- **Капсула:** DIY-AI-база + свой финиш → ~$50–150 фрилансер-финиш → ~$300 человек только если не прошло (fallback, не дефолт).
- **Голос:** AI-PA, чистые лицензии (ElevenLabs paid / Chatterbox / Kokoro / Piper; НЕ Coqui XTTS-v2), PA-радио-финиш, гейт-эксперимент.
