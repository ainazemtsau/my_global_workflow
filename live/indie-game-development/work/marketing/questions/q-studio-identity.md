# Вопрос: имя студии + один кросс-игровой ник (ЗАМОРОЖЕНО 2026-06-17)

**Тип:** DECISION. Решено внутри процесса: ресёрч (нормы именования + проверка доступности/коллизий по 17 кандидатам, 2 прохода, 21 агент) + вкус/выбор владельца. Имя студии ≠ имя игры/мира (последнее — канон g-d3a8); лор не выдуман.

## РЕШЕНИЕ

- **Имя студии: Chinvat** (голое слово; идентичность = и есть ник).
- **Кросс-игровой ник: `chinvat`** (имя == ник; один хэндл на все каналы; строчными, чистый ASCII, 7 символов).
- **Форма:** голое «Chinvat» как идентичность. Суффикс «Games» — опция ТОЛЬКО для Steam-поля Developer (поисковая различимость); решается в q-capture-setup, не здесь.
- **Смысл / трассировка к вводным владельца:** мост Чинват из космологии Заратустры — порог-переправа над бездной, суд при переходе «на ту сторону». Несёт глубину / опасную экспедицию / переход; **НЕ залочено на газ** → переживёт первую игру и встанет на любую будущую (студия = переиспользуемый seed, устав). Владелец тянулся к мотиву «Так говорил Заратустра» (слом устоев игр / новый опыт / «работаем глубоко»); Chinvat несёт это ощущение, оставаясь произносимым ownable-словом.
- **Почему НЕ громкое заявление:** имя не выпячивает «мы революция / мы глубокие». У неизвестной студии громкое имя выписывает чек, который игра ещё не обналичила (риск претенциозности и «умрёшь со стыда через 10 лет» — норма-ресёрч). Глубину зарабатывает геймплейный клип, не вывеска; кто полезет гуглить «Chinvat» — найдёт мост над бездной (показать, не сказать). NB: ось «мы никто» = объективная СТАРТОВАЯ ТОЧКА (нет аудитории/продаж/имени), НЕ бренд-характеристика для продвижения (уточнение владельца 2026-06-17 → knowledge/marketing-we-are-nobody-is-starting-point).
