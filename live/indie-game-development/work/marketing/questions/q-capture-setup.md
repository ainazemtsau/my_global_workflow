# Артефакт: поднять слой захвата — страница + рассылка + имя (СЕЙЧАС)

Порождён q-foundation. «Чистый апсайд, поднять сразу»: Steam Coming-Soon страница, рассылка «бета-ключ за email» (opt-in, без спама), зарезервированный постоянный хэндл студии. Капсула — через q-ai-pipeline.

## Текущее состояние — checkpoint 2026-07-12

- Готово: студия `INOMAND`; домен `inomand.com`; регистрационная входящая почта `accounts@inomand.com`; Cloudflare 2FA + recovery; реальное письмо доставлено.
- Готово как решение, но ещё не захвачено: display `INOMAND`, единый хэндл `@inomandgames`.
- Steamworks проверен: приложений нет, один уже оплаченный app credit доступен.
- **Не делать:** не создавать Steam App до отдельного решения по названию игры. `Chinvat` отвергнут как имя студии и не является названием игры.
- Не готово: публичные аккаунты, opt-in рассылка, Steam AppID/Coming-Soon page, финальная капсула и реальные store screenshots.

## Что сделать (форжим внутри)
Захватить студийные аккаунты `INOMAND / @inomandgames` без публикаций → после отдельного выбора названия игры создать Steam AppID → завести страницу-болванку → рассылку с opt-in → связать «вишлист/email» в воронку до первого поста. Логистика делегируема AI/помощникам; публичные действия и голос требуют owner-гейта.
