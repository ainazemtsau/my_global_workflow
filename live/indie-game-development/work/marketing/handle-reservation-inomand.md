## YouTube — пошаговая инструкция (Brand-канал под существующим Google)

Модель: ОТДЕЛЬНЫЙ канал (Brand Account) под уже существующим личным Google. Новый Google-аккаунт, SMS и пароли НЕ нужны — наследует защиту личного аккаунта. Позже в Brand-канал можно добавить менеджеров (помощников) без выдачи личного пароля.

Значения для вставки:
- Name (название канала): `INOMAND`
- Handle (хэндл, вводить без «@» — поле подставит само): `inomandgames`

Шаги:
0. Открыть youtube.com; убедиться, что залогинен под нужным личным Google (аватар справа вверху).
1. Открыть список каналов: `https://www.youtube.com/channel_switcher`. Fallback: аватар → Настройки → Аккаунт → «Добавить или управлять каналами».
2. Нажать «Создать канал» / «Создать новый канал» (Create a channel / Create a new channel). Это создаёт отдельный Brand-канал, НЕ переименовывает личный.
3. В окне: имя → `INOMAND`; хэндл → `inomandgames`. Зелёная галочка = свободен. Красное/«занят» = СТОП, не создавать, эскалировать владельцу (единый хэндл — owner-решение, не подбирать замену самовольно). Аватар пока пропустить.
4. Нажать «Создать канал» (Create channel).
5. Проверить: канал `INOMAND`, адрес `https://www.youtube.com/@inomandgames`.

Ничего не публиковать — канал остаётся пустым. `accounts@inomand.com` для YouTube НЕ используется (нужен для сервисов со свежим аккаунтом: Reddit/X/Bluesky и т.д.).
