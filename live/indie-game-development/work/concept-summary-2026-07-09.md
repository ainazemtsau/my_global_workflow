## ЧТО ПРИНЯЛИ — ядро игры

**1. Структура из трёх фаз** (твоя, подтверждена несколько раз)
- **Фаза 1 — тихо:** газы спят, почти невидимы; бригада ходит с приборами и разбирается, ЧТО и ГДЕ лежит (разведка-детектив).
- **Фаза 2 — движение:** газы просыпаются (тепло / перепад давления / скорость / забор / таймер), текут, слоятся, реагируют друг с другом, влияют на игроков.
- **Фаза 3 — эскалация (опц.):** этаж идёт вразнос, финал-бегство.

**2. Спящие газы + разведка** — принято как вход. Сон оставляет следы (иней, осадок, звук, дыхание вентиляции); нет следа — нет газа, игра не жульничает. «Сон» — это ДИЗАЙН-состояние (дремлет как монстр), а не движковая оптимизация байт-сна (важная поправка владельца).

**3. ЯДРО ДОБЫЧИ: «ПУЗЫРЬ»** — главный глагол игры (принято: «мне нравится»)
- Кладёшь мембранное кольцо на спящее газовое озеро → плёнка надувается ПУЗЫРЁМ, внутри — живой видимый газ (не железный баллон с циферблатом; газ остаётся звездой всю дорогу).
- Несёшь пузырь наружу целым. Пузырь ведёт себя как его газ: тяжёлый — медбол, катится/волочится; лёгкий — рвётся вверх, несёшь на привязи; «Сквозняк» — тянет к дверям; «Хор» — пульсирует; боится жара / искр / удара / ударной волны.
- Лопнул — газ не исчез, он ВЫШЕЛ здесь, проснувшийся и злой → дальше вся симуляция. Провал не показывает экран — выпускает монстра в комнату.
- Один глагол «неси, не лопни» — казуалу объяснять нечего (интуиция воздушного шарика с детства). Наш аналог «переноса» из R.E.P.O., но груз — живой кусок мира, а не мёртвый предмет.

**4. Большое облако = берёшь ЧАСТЬ** (последний принятый уточняющий ответ в рамках пузыря)
- Кольцо — ковш с потолком объёма, не пылесос на всё озеро. Сам решаешь, сколько надуть: мелкий (лёгкий, дёшев, влезает в дверь) ↔ огромный (дорогой, вдвоём, в дверь НЕ лезет — грузовой коридор / пролом).
- Что не влезло — остаётся, озеро на глазах мелеет (масса честная). Остаток = следующая ходка = «ещё раз», встроенное в саму механику без квот.
- Центр озера плотный/дорогой, край разбавлен; забор тревожит остаток (эскалация от рук, не по таймеру).

**5. Бестиарий-подход** — газы это монстры БЕЗ мозгов, каждый = одно детерминированное правило физики (Тихоня / Сквозняк / Вспышка / Разлив / Руки / Хор — иллюстрации, не финальный ростер). «Руки в газе» приняты; «глаза» отвергнуты (рисуют лицо → существо → ломают весь фокус «зверь без ума, но с законами»).

**6. Опасность и распространение** (из фактов движка)
- Бесконечного распространения нет: масса сохраняется, конечный склад этажа, разбавление (широко-слабо ↔ узко-смертельно), ёмкость комнат.
- Палитра вреда: душит (нет кислорода) / травит (доза) / взрыв (телеграф→волна) / мороз-ожог / слепит и глушит голос / тащит. Спящий ≈ безвреден, разбуженный опасен, разбуженный не вовремя смертелен.

## ЧТО ОТВЕРГЛИ (и почему — это компас)

- **«Вынести хрупкий предмет»** (курьер-квест) — газ снова декорация-погода; добывать надо сам газ.
- **«Комната-лёгкое»** (помпа + заделай щели) — оборонительные глаголы, работа сантехника; скучно, «сам бы не играл».
- **«Правильная катастрофа»** (дирижируй столкновениями + лови ампулами продукт) — «кинь ловушку туда, где телеграф» казуалу не объяснить; за уши; каждый забег одинаков.
- **«Глаза у газа»** — лицо → существо → ломает «монстр без мозгов».
- **Мета «Вахта»** (квота-цикл) — generic-каталог чужих механик, не выведено из нашей игры.

## ЧТО ОТКРЫТО / НЕ РЕШЕНО

- **Мета-петля.** Стоящее предложение — **«Лёгкие»** (база = лёгкие бригады; две рукоятки ДЫШАТЬ / ПРОДАТЬ; воздух = валюта; долг-кабала как мягкий ресет). Не принято и не отвергнуто — фокус ушёл на ядро-глагол. Пузырь + «остаток = ходка» уже даёт «ещё раз» и без формальной меты.
- **Рамка ядра** (`d-ono-dyshit-core-frame-fork-001`, Design Lab): capture-first / mission-value-first / гибрид — официально ОТКРЫТА. Пузырь — это capture-first механика, то есть чат фактически выбирает capture-first. **Это надо СВЕСТИ с Design Lab, а не заморозить здесь.**
- **Точные числа / ростер / инструменты / экономика / урон** — вниз по дороге (урон = слайс Sc-damage).

## КОМПАС: чему научили провалы (правила для ЭТОЙ игры)

1. Ядро — это ГЛАГОЛ (неси), а не ПРОЦЕДУРА (ставь-качай-заделывай).
2. Объясняется одной фразой из детской интуиции; казуалу нечего учить.
3. Повторяемо каждый забег и не надоедает; вариативность — от этажа × типов × сколько разбудили, а не от новых правил.
4. Активные глаголы, не оборонительные; фан — в физическом действии, понятном всем.
5. Газ — видимая ЗВЕЗДА, не спрятан в прибор / циферблат.
6. Провал возвращает газ в мир, а не показывает Game Over.
7. Хороший глагол сам тянет ВЕСЬ наш сим (типы, реакции, слои, пролом, распространение) без «впихивания».

## ЧТО ИССЛЕДОВАНО И ОТЛОЖЕНО (фон)

- 7-агентное исследование (психология хуков, рынок, альтернативы: hazmat-служба, вынос людей, атмо-кухня, один вечный завод) — по решению владельца отложено (интересует концепт, не маркетинг). Хуки, которые подтвердились и легли на пузырь: push-your-luck (сколько надуть / ещё ходка), near-miss (донёс на волоске), спасение физикой, мастерство (что брать и каким маршрутом).

## ПРОВЕРКА (как узнаем, что работает)

Первый грейбокс на двоих, без меты: голое ядро «надуй пузырь — вынеси» должно само дать четыре «да» — несут вдвоём и координируются; спорят, сколько брать / идти ли ещё; про газ говорят «ОН»; незнакомец понимает 30-сек клип без слов. Мета проверяется вторым грейбоксом.
