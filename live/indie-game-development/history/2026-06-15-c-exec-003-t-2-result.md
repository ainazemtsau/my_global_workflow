## outcome

t-2 proved the **real game world streams host→followers and stays in lockstep-exact sync over a
player-hosted seam**, against its frozen `done_when`:

- **Multi-layer fabric (≥2 layers).** Gas registers as `LayerKey 0` over the v1 `FieldState`;
  temperature registers as `LayerKey 1` over a SIBLING `int[]` (never a 9th `ZoneRecord` int).
  A real layer fabric drives them: `LayerRegistry` (ascending-`LayerKey`, no dict enumeration) +
  a separate `RevisionFeed`/`IGridEventBus`; single-writer-per-`(LayerKey,phase)` enforced by a
  registry-minted token (duplicate registration throws); pinned phases
  `P0 commands/breach → P1 layer-sim → P2 cross-layer events → P3 publish`. Adding temperature
  touched no gas file and consumed zero gas RNG (proven by an RNG-conservation guard).
- **Contractive, quiescible gas sim.** A host-only integer, mass-conserving room-graph diffusion
  with strictly-decreasing total-variation → non-empty fixed point → dirty-chunks → 0 (so a settle
  is real, not a false flatline).
- **Dual lossless⇄lossy consistency through ONE codec.** Lossless = the same codec at `Q=1`,
  deadband 0. Run A (lossless): per-tick host-vs-each-client reconstructed cell-hash **bit-exact
  every tick over the whole run including the breach**. Run B (lossy, deadband=Q=16 + per-client +
  aggregate clamps): `|client−host| ≤ Q` **every tick** AND **bit-exact at every converged settle**
  via a FULL-INT32 residual flush (not a quantized emit). Kill-gate domain = reconstructed FIELD
  CELLS only (pinned `Fnv1a64` fold).
- **Real controlled breach over the lockstep input plane.** A breach is a `TickInput.Command` on
  `ITickInputBus`, released by the input-complete barrier so all peers agree the tick; the host flips
  one `RoomGraph` portal Closed→Open at `breachTick` and gas then diffuses across it. Field state was
  NOT retrofitted into the input bus — only the breach command rides it.
- **Three flow controls engaged together.** Deadband + per-client (150 KB/s) + aggregate host
  (275 KB/s), each proven ENGAGED by a negative over-budget test (a silently-disabled control fails).
- **Deterministic fault tolerance.** A test-side drop+delay channel decorator; the revision barrier
  never applies a stale (lower-revision) chunk (proven by an INDEPENDENT host-accepted-revision
  oracle) and recovers to bit-exact at the next converged settle.
- **Honest measurements + frozen reproducibility.** Machine-readable lossless+lossy wire artifacts and
  committed hashed scene/repro JSON.
- **Invariants held:** engine-free core (0 Unity refs, R13), networking edge-only (R12/R14),
  deterministic (no wall-clock, seeded RNG only). ADR-0003 v2 C1–C22 frozen and untouched by the build.

This de-risks the central bet question — *can a rich shared world stay perfectly synchronized over the
internet, player-hosted, cheaply and robustly* — at the building-block level.
