  по ФИЗИКЕ порога, не дирекции (верх склады/админ+жильё/калибровка / глубь забои-переработка-опытные
  ячейки, только работа / дно терминус); **глубь горяча спустя годы = экзотермия порогового режима +
  рукотворная сгущёнка/запертость** (печи мертвы, «тепло недр» только генезис; газ отражает условия).

  падает с глубиной); всё АНАЛОГОВОЕ (циферблаты/рычаги/бумага, без экранов) = «прибор» видим + регистр B;
  «горизонты = строки замеров» (калибровочные сектора рядами по глубине, жильё = гигиена замера); масштаб
  (короткий верх → длинная глубь с ангарами → одна каверна); видимая тепловая подпись по поясам.
- **Газ на уровнях:** вход в основном ЧИСТ, опасность РОЖДАЕТСЯ карманами из источников симуляцией
  (не туман, всегда структура; «фон» = тонкий безопасный субстрат); глубина = энергия активации (выше
  засев плотности/темп → тот же порог легче; множит лёгкость/интенсивность, НЕ покрытие; порог не
  параметризован глубиной); холод усмиряет → тепловая география = ресурс; текущее состояние = level-gen
  начальные условия, разлив-вверх = бэкстори засева (конвекция, не runtime-сим).
- **Исчезнувшие этажи:** погибшие глубокие этажи, стёртые конторой из бумаг (карта врёт глубже уровня) =
  открытая дверь вглубь (анти-LOST), но разгадка ГЛУБЖЕ, в терминусе; стёртые этажи ведут СКВОЗЬ себя; не
  детектив-механика, фон; всё ещё врубленное пространство (шахта-спина сквозь них → последний природный ход).
- **Перемещение/кооп-кости:** спуск непрерывный, спина = шахта лифта (и канал газа; назад через злую
  глубь); гейт глубины = САМА СРЕДА под другой режим, не замки/квоты (добровольная ставка); только кости
  коопа (большие разнесённые залы + не-ко-локальность разносят игроков; механизм → q-coop-shape).
- **Высота/наследие:** мягко отложено (стиль→q-visual-style; снаряга/добыча/груз/цикл/число-этажей/
  форма-ставки→q-coop-shape/q-gas-families/shape; терминус→q-the-deep); газ без воли, sim-derived, grounded,
  дно достижимо, кооп не соло, история фон.
