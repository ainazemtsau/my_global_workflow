Канон-ядро (14 инвариантов, 6 групп):
- **Вещество:** одно вещество — много лиц; ЛИЦО/режим = **непрерывная** f(плотность×температура×
  соседи×катализаторы) = «зеркало среды»; семьи-архетипы = режимы одной функции. Газ выставляет
  **ПОЛЕ** (интенсивность+категория), сам НЕ пишет в чужое состояние — урон из замороженного перечня
  (горит/морозит/корродирует/вытесняет O₂), «меняет пространство» ЧИТАЮТ другие системы, стойкий след =
  коррозия через consequence-событие. Превращение = смена ТОЖДЕСТВА (пороговая реакция → новый газ).
- **Реакции:** из малого ФИКСИРОВАННОГО числа осей, не N² (новый газ любого тира = НОЛЬ новых попарных
  правил, поверхность O(осей)); один свод правил — два вида (реакция ⇄ рецепт формирования).
- **Пространство+кооп:** газ = поле со СТРУКТУРОЙ (карманы, не туман), читаемость обязательна; эффекты
  **НЕ ко-локальны** = структурный ШОВ кооперации (типизированное событие proximity-independent, сам
  газ доезжает транспортом); рождение облаков — симуляция из начальной сцены level-gen, не скрипт.
- **Глубина+тиры:** глубина рабочей зоны флипает РЕЖИМ **температурой** (вид угрозы, не число; порог НЕ
  параметризован глубиной; плотность = интенсивность; предельная плотность+холод = инертность);
  **«глаз бури»** (терминус холодный — заморожено q-world-origin #7); мелкая тир-лестница = добыча+
  двигатель (ценность = f(тир×усилие-смешивания×условия)), высшие тиры = эмерджентные жуткие формы.
- **Граница жути:** воспринимаемая агентность flagship ОБЯЗАНА оставаться полевой физикой (движение =
  деривация диффузии, без таргетинга/преследования; «гонится» отклоняется при регистрации).
- **Экономика:** игрок добровольно крутит риск↔награду (НЕ крафт-духота), ось СВЯЗАНА через ОБЩЕЕ поле.
