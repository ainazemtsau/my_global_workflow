# Health AI Core is a guidance/governance layer, not only a data store
accepted: 2026-06-13   read_by: shape/review for health AI, nutrition, and training/activity nodes before creating tasks   status: current

The verified core slice showed that useful cross-chat health operation needs source-of-truth boundaries plus guidance/governance: current phase, next actions, investigation queue, procedures, roles, pending patches, and escalation rules.
Direction OS remains strategic only. Raw daily food, training, activity, check-in, photo, or app-export data stays in Health AI System or future integrations; Direction OS receives only summary, problem, decision, incident, evidence pointer, or CALL.
