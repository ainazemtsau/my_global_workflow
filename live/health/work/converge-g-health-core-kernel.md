### §KW-AF — anti-fragility / extension contract (R6)
- **KA1** (sys) The atomic isolation unit = one procedure = one bounded function per term; "fix" = edit one
  procedure file, "replace" = swap one — no edit to siblings or the router. →R1/R2/R3/W53/W54
- **KA2** (imp) "Independent" = three concrete edit-shapes with bounded diffs: ADD a domain = 1 folder + 1 registry
  line; REMOVE = delete its x_<domain>_* + its 1 line; FIX = edit one procedure file. No core edit. →W57/W54/W58/WA8
- **KA3** (imp) A procedure's blast radius is bounded by namespace+ownership: writes only its own x_<domain>_*
  fields/folder; reads core concepts but never widens/mutates core; never writes another domain's namespace. →W58/W59/CA4
