# No Single Track Collapse Eval

status: active_eval

## Purpose

Validate that multi-dimensional Direction success is not collapsed into one comfortable work track without explicit decision and trace.

## PASS checks

- Multi-dimensional goal tracks or success dimensions are reviewed.
- Deferred dimensions have trigger and reason.
- Active Front and Work Graph preserve trace to touched success dimensions.
- Selection explains evidence, risk, dependency, capacity, and reversibility tradeoffs.

## WARN checks

- One dimension dominates temporarily, and deferred dimensions have explicit review triggers.
- Some success dimensions lack current evidence but are visible as unresolved.

## FAIL checks

- Complex Direction collapses to one comfortable track without explicit decision.
- Deferred dimensions disappear from graph/map/front trace.
- Work Graph ignores success dimensions touched by the Active Front.

## Required recovery/repair action

Stop front/work expansion, restore success-dimension trace, and require Active Front or Direction Map repair before continuation.
