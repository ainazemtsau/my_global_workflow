# Work Graph Formation Runbook

status: active_formation_runbook

## Trigger

Use after an accepted or explicitly admitted candidate Active Front needs the minimum local Work Graph to close its exit criteria.

## Inputs

- Active Front and exit criteria;
- source map areas and constraints;
- evidence gaps, dependencies, blockers, and candidate nodes;
- return destination.

## Source reads

Read the Active Front source, current status, accepted/candidate map references, and any existing local front workspace if it exists.

## Formation steps

1. Confirm the target is Work Graph only.
2. Classify accepted front evidence, candidate context, and unresolved gaps.
3. Frame what the graph must close for the Active Front.
4. Generate 2-3 node graph options or sequencing shapes unless blocked.
5. Define criteria: minimum proof path, dependency clarity, evidence yield, node boundedness, scope control, and closure usefulness.
6. Identify evidence requirements and failure conditions.
7. Attack backlog expansion, global roadmap drift, hidden product execution, and false dependency chains.
8. Cut non-nodes, future backlog, and polish work.
9. Draft the minimum Work Graph.
10. Record rejected/deferred graph shapes and cut items.
11. State read limitations.
12. Ask the acceptance or launch question.
13. Close the event loop.
14. Provide exact next move, usually Work Contract formation for one node.

## Must include

- minimum nodes needed to close the front;
- dependency structure;
- proof path;
- explicit non-nodes / cut items.

## Must not do

Do not become global Direction Map, backlog, roadmap, or Action Inbox.

## Outputs

Return a candidate Work Graph compatible with `workflow_v3/templates/WORK_GRAPH_TEMPLATE.md`, with evidence, risks, cuts, acceptance question, Event Loop Closure, and exact next move.

## Acceptance boundary

Persistence or state mutation requires the applicable acceptance/update path.

## Stop conditions

Stop if Active Front is missing, the graph would span multiple fronts, or the node list is a disguised backlog.
