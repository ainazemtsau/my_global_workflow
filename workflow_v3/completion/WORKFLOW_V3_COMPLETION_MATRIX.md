## Removed Authority

The active architecture has no separate start protocol, before-action protocol, or old operation-surface contract. Runtime authority is consolidated in `CHAT_LIFECYCLE_PROTOCOL.md`.

## Completion Rule

CHECK compares actual result to the selected procedure completion contract. No global fixed list of done/result types is defined here. Handoff, package, card, check, storage, copy-paste, and dependency-call artifacts are not parent completion by themselves.
